## 🎨 项目哲学

### 牧羊人隐喻

```
┌─────────────────────────────────────┐
│         人类（牧羊人）               │
│    决定做什么、怎么做、做到什么标准   │
└─────────────┬───────────────────────┘
              │
              ▼
┌─────────────────────────────────────┐
│         牧羊犬（AI监督者）          │
│    分配任务、检查质量、确保安全      │
└─────────────┬───────────────────────┘
              │
              ▼
┌─────────────────────────────────────┐
│         羊（AI执行者）              │
│    编写代码、运行测试、生成文档      │
└─────────────────────────────────────┘
```

### 核心原则

1. **人类始终处于决策链顶端**
2. **AI被驯化而非被取代**
3. **激励对齐而非强制控制**
4. **安全优先于效率**
5. **责任归属明确**
